This calculator has been tested in "Medium Phone API 36.0 (anroid-x64 emulator) and Windows (windows-desktop).

Some hints:

Type `PLOT(SIN, 0, 2*PI)` followed by `ENTER` to plot a `SIN` (SINE).

Type `PLOT(x=>x^2, 0, 10)` followed by `ENTER` to plot `x^2` in the interval `0` to `10`.

Press `LOAD` and select `calculator.shql` to run a calculator program. On Android the programs will be copied to the Downloads folder. 

If a program hangs, press `BREAK` to abort execution.

Type `CLS` followed by `ENTER` clear the terminal or `HIDE_GRAPH` followed by `ENTER` to hide the graph from the plot function.
